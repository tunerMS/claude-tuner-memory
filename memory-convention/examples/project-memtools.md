Пример заметки типа **project**. memtools (`~/.claude/memory-tools/`) — надстройка
над файловой памятью: семантический recall, протухание/дубли, авто-линковка,
опциональный граф знаний.

**Как пользоваться:** `mem recall "<запрос>"` — top-k релевантных кусков по всей
памяти; используй вместо чтения больших файлов целиком.

Связано: [[project-session-learning]] (обучаемые инстинкты поверх тех же сессий).
